iPipet



USAGE:

Optimizing protocols for liquid-handling robots can be tedious for even simple tasks.

iPipet was originally developed for combinatorial pooling of hundreds of samples as an alternative to costly robots.

We decided to make the program available to others who might have similar struggles.


iPipet can be used to transfer samples between plates.

One example may be cherry picking samples from multiple "source" plates to a common "destination" plate.

It can also be used to illuminate wells of a single plate, ie. to dispense reagents or normalize samples.

GETTING STARTED:

1. If you haven't already, check out our demo using a sample CSV file.

2. Otherwise, the first thing you need to do is create your own CSV file. This file must be correctly formatted.

The top plate on the screen (see above image) is the "source" plate and the bottinom plate is the "destination" plate.

The well-numbering is incredibly important: wells must be numbered 1-96. Wells 1-8 correspond to wells A1, B1, C1 ... H1 NOT A1, A2, A3 ... A8.

If your wells are not in this order, an easy solution is to sort the well columns in Excel so that they are in the correct order and then assign the appropriate number.

(Note that the header will be skipped so as long as the column order is correct you can name the columns whatever you'd like. Just make sure you do include a header in the file.)



To light up destination plate wells only, simply enter place holders in the source plate and source well columns (or vice versa if you prefer) in the CSV file:

Note that the volume column is empty here. If you do not need to pipet variable volumes, simply leave this column blank. If you do use the volume column, ALL cells must have a value. Simply use "0" as a placeholder if necessary.

3. Once your CSV file is ready, go ahead and give your project a name and upload it on the homepage.

4. You can provide an email if you'd like iPipet to send you the project link. Otherwise, feel free to leave this field blank. Just be sure to save your unique link. (It's easier to upload the file on a computer and open the link on your tablet.)

5. Click "SUBMIT CSV" at the bottom of the homepage. Then just make sure the wells of each plate line up with the plate diagram.

(A layer of plastic wrap over the tablet screen works well to prevent the plates from moving too much and still works with gloves.)


6. It is highly recommended to test the program before handling actual samples to be sure the design is correct.

7. Make sure you have all your plates and you're ready to go!

Happy pipetting!
